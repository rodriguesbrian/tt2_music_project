# TripleTen2_Music_Project

Este projeto analisa dados de um serviço de streaming de música para testar uma hipótese sobre o comportamento dos usuários em duas cidades fictícias: Springfield e Shelbyville.

---

## 🧪 Hipótese

A atividade dos usuários é diferente dependendo do dia da semana e da cidade.

O objetivo do projeto é aceitar ou rejeitar essa hipótese com base nos dados disponíveis.

---

## 🔍 Descrição do Projeto

O projeto está dividido em três etapas:

1. **Etapa 1 – Exploração de Dados:**  
   Visualização inicial e análise das observações para entender a estrutura e padrões do dataset.

2. **Etapa 2 – Pré-processamento:**  
   Limpeza e preparação dos dados para análise estatística.

3. **Etapa 3 – Teste de Hipótese:**  
   Aplicação de técnicas de análise para testar a hipótese proposta, seguindo passos de programação detalhados.
